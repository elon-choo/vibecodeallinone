## Grade Criteria

| Grade | Score Range | Description | Action |
|-------|-------------|-------------|--------|
| A+ | 95-100 | Excellent | Merge immediately |
| A | 90-94 | Very Good | Merge with minor notes |
| B+ | 85-89 | Good | Quick fixes then merge |
| B | 80-84 | Acceptable | Address warnings |
| C | 70-79 | Needs Work | Significant changes |
| D | 60-69 | Poor | Major revision needed |
| F | < 60 | Failing | Rewrite required |

## Usage

```typescript
function generateReviewReport(reviewResult: ReviewResult): string {
  const template = loadTemplate('review-report-template.md');

  return template.render({
    date: new Date().toISOString(),
    file_count: reviewResult.files.length,
    lines_added: reviewResult.linesAdded,
    lines_removed: reviewResult.linesRemoved,
    score: reviewResult.score,
    grade: calculateGrade(reviewResult.score),
    // ... other fields
  });
}
```
