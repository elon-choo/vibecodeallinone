Skills

12 skills across 5 layers.

Persistent knowledge graph for cross-session memory, 12 enforcement skills for consistent quality, and auto-trigger hooks for a unified workflow. Each skill targets a specific aspect of code quality — together, they form a comprehensive quality gate.

Foundation Layer
📈
codebase-graph

In-memory code knowledge graph. Maps functions, classes, imports, and call chains for instant codebase understanding.

🎯
smart-context

Token-efficient context assembly with 72-77% savings. 6 detail levels from minimal to deep analysis.

🔃
graph-loader

JSON-to-Neo4j graph loader. Bridges the in-memory graph with persistent Neo4j storage for cross-session memory.

Analysis Layer
💥
impact-analyzer

Change propagation analysis and risk scoring. Know exactly what breaks before you push.

📑
arch-visualizer

Auto-generates Mermaid and PlantUML architecture diagrams from your actual code structure.

Quality Layer
💎
clean-code-mastery

SOLID, DRY, KISS, YAGNI + OWASP enforcement across 8 programming languages. The backbone of quality.

🛡
security-shield

OWASP Top 10 scanning with 40+ secret detection patterns. Catches API keys, tokens, and credentials before commit.

✅
tdd-guardian

TDD workflow enforcement with fake-test detection. Ensures tests are meaningful, not just passing.

Validation Layer
👁
code-smell-detector

Detects 22 code smell patterns including god classes, long methods, feature envy, and shotgun surgery.

📄
naming-convention-guard

Per-language naming rules. Enforces camelCase, snake_case, PascalCase, and more based on language conventions.

Integration Layer
🎵
vibe-coding-orchestrator

The conductor. Coordinates all 12 skills across 9 layers for seamless, automated code quality management.

🔍
code-reviewer

Quality gate scoring with multi-dimensional analysis. Generates actionable review reports with fix suggestions.
