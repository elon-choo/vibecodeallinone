# Claude Code Power Pack

12 AI skills + Knowledge Graph MCP server for comprehensive code quality.

- **clean-code-mastery**: SOLID/DRY/KISS enforcement + OWASP security for TypeScript, Python, Go, Rust, Java, C++, C#, Kotlin
- **security-shield**: OWASP Top 10, 40+ secret detection patterns, input validation
- **tdd-guardian**: TDD workflow, fake-test detection, 80% coverage gate

### Foundation Layer
- **codebase-graph**: In-memory code knowledge graph with dependency analysis
- **smart-context**: Token-efficient context extraction (72-77% savings)
- **graph-loader**: JSON-to-Neo4j graph loader

### Analysis Layer
- **impact-analyzer**: Change propagation analysis with risk scoring
- **arch-visualizer**: Mermaid/PlantUML architecture diagram generation

### Validation Layer
- **code-smell-detector**: 22 code smell patterns with severity scoring
- **naming-convention-guard**: Language-specific naming enforcement

### Integration Layer
- **code-reviewer**: Quality gate scoring with unified review
- **vibe-coding-orchestrator**: Coordinates all skills across 9 layers

## Usage

Type these in your AI coding session:
- `vibe review` — Full code review
- `vibe init` — New project setup
- `vibe secure` — Security scan
- `vibe pipeline` — All quality stages
